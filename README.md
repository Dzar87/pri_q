# PriQ

Priority queue that uses a binary heap for sorting the priorities.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'pri_q'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install pri_q
